User Manual

Welcome to the Machine Learning Model Training and Prediction Tool. This manual will guide you through the features and functionalities of the website.

Uploading a CSV File

To begin, upload a CSV file containing your dataset. Click the "Upload CSV File" button, select your file, and click "Upload".

Selecting Features and Target Column

Choose the features you want to use for training the model from the "Select Features" dropdown. Select the target column from the "Target Column" dropdown. These selections are crucial for training the model.

Training the Model

Select the type of model you wish to train from the options provided. Below is a brief description of each model:

Linear Regression

A simple model that assumes a linear relationship between features and the target variable. It predicts the target by finding the best-fitting line through the data points.

Random Forest

An ensemble method that uses multiple decision trees to improve prediction accuracy and control overfitting. Each tree is trained on a random subset of the data and features, and the final prediction is an average (for regression) or majority vote (for classification).

Gradient Boosting

Builds trees sequentially where each tree corrects errors made by the previous ones. This method often provides better performance than individual models by focusing on difficult cases.

Decision Tree

A model that splits the data into subsets based on feature values, creating a tree-like structure for prediction. It’s easy to understand and interpret but can be prone to overfitting.

Support Vector Regressor (SVR)

Finds a hyperplane that best fits the data within a certain margin of tolerance. It is useful for high-dimensional spaces and can handle non-linear relationships using kernel functions.

K-Nearest Neighbors (KNN)

Predicts the value based on the average of the k-nearest neighbors' values in the feature space. It’s simple but can be computationally expensive for large datasets.

AdaBoost

An ensemble technique that combines multiple weak learners to create a strong learner. It adjusts the weights of incorrectly classified instances to improve model performance.

Lasso Regression

A type of linear regression that includes L1 regularization to prevent overfitting by shrinking some coefficients to zero, effectively performing feature selection.

Ridge Regression

Similar to Lasso but uses L2 regularization to penalize large coefficients without setting them to zero. It helps in handling multicollinearity.

Elastic Net

Combines L1 and L2 regularization to enhance the model's performance and stability. It is useful when there are correlations among features.

Extra Trees Regressor

An ensemble method that creates multiple decision trees and aggregates their predictions. It can improve model performance by introducing randomness in the training process.

Histogram-Based Gradient Boosting

Efficient gradient boosting using histograms to split data, improving computation time and performance. It’s particularly useful for large datasets.

Providing Example Inputs

To obtain predictions from your trained model, provide example inputs in the designated fields under "Example Input". Ensure that all fields correspond to the selected features. Click "Get Prediction" to see the prediction result.

Exporting the Model

After training, you can export the model as a pickle file. Click the "Export Model" button to download the trained model file.

Exporting Encoders and Scalers

If you need to use the same encoders and scalers for future datasets, click the "Export Encoders & Scalers" button. This will download a pickle file containing all the encoders and scalers used during preprocessing.

Handling Errors

If you encounter errors such as "Unexpected categorical input" or "Prediction error", ensure that your input values are valid and match the expected data types. Verify that all categorical inputs are known to the model and that numeric inputs are properly formatted.
